# ai-handoff.md Structure

## Current Objective

Single highest-priority goal.

## Current Status

Short summary of where work currently stands.

## Active Tasks

| Task         | Status      | Notes   |
| ------------ | ----------- | ------- |
| Example Task | In Progress | Details |

Valid status values:

- Not Started
- In Progress
- Blocked
- Testing
- Complete

## Current Focus

Detailed explanation of what is actively being worked on.

## Recent Progress

Work completed during recent sessions.

## Files Recently Modified

For each file, record:

- File path
- Purpose
- Relevant notes

## Important Discoveries

Temporary findings that may affect current work.

## Blockers

For each blocker, include:

- Blocker Name
- Description
- Impact
- Potential Resolution

## Testing Status

- Completed Tests
- Pending Tests
- Failed Tests

## Commands and Procedures

Important commands used recently.
Example:

```bash
example-command
```

## Next Actions

Ordered list of recommended next steps:

1. First action
2. Second action
3. Third action

## Risks

Current risks and concerns.

## Questions Requiring Answers

Open implementation questions.

## Recommended Starting Point for Next Session

Provide a concise summary:

- What was completed
- What remains
- Where to resume work
- First recommended action
